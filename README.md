# ⚖️ LawBot – AI Legal Research Assistant

LawBot is an AI-powered legal research and drafting assistant that helps users analyze legal documents, perform legal research, and generate legal content using modern AI models.

## Features

* 🤖 AI-powered legal assistant
* 📄 Upload and analyze legal documents
* 🔍 Legal research support
* 📝 Draft legal notices and contracts
* 💬 Interactive chatbot interface
* ⚡ Fast React + Vite frontend
* 🎨 Modern responsive UI

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* Shadcn UI

### Backend

* Python / FastAPI (if applicable)
* AI APIs (OpenAI / Gemini / Claude / Kimi, as applicable)

## Installation

Clone the repository:

```bash
git clone https://github.com/Anmolkankarwal/LAWBOT.git
cd LAWBOT
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
